Controle de acesso

Visualizar permite consultar o módulo. Gerenciar permite cadastrar, editar, movimentar e excluir registros.

O sistema impede excluir o usuário conectado e também impede remover o último administrador ativo.

Atenção: esta versão salva usuários e senhas codificadas somente no navegador. Para produção, utilize backend, banco de dados, senha com hash forte e autenticação centralizada.
